MiMi Posted on 2010.07.21 10:22 // 22 Comments

哇哈哈，谁说 groupoffice 只能安装到 Linux 环境，我就在Windows系统安装成功了，版本为3.5.11。
折腾了一上午，重写了files模块一半的代码，终于成功了，难掩兴奋之情，臭美一下。
有时间我会把详细过程记录下来。

  6. MiMi Says:
    2010.09.04 11:56

    楼上的问题我碰到过，并且已经解决，
    根本原因还是files模块无法创建用户目录导致的，
    当登陆groupoffice时，系统无法自动创建用户的文件夹，脚本找不到相应目录无法执行，前端ExtJs又不会显示错误信息，页面才会总是显示“载入基本系统…”等待响应，
    建议重新安装且安装的时候不要选择中文，因为在中文语言包下，管理员目录对应的是中文文件夹，
    如果还是不能登录系统，请手动根据登录脚本创建所需文件夹或修改files模块。
