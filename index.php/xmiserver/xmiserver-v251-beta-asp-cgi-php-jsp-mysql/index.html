简介：
XmiServer_v2.5 是一款基于 Apache+Tomcat 的全能服务器组合套件，
拥有安装简单、便捷、绿色、免配置等特点，
可以快速搭建 ASP+CGI+PHP+JSP+MySQL 全能网站服务器平台，
方便您调试和开发ASP、CGI、PHP、JSP 等程序。

    XmiServer为了调试方便，开启了E_STRICT常量，您可以编辑 php.ini 文件
    将 error_reporting = E_ALL & ~E_NOTICE | E_STRICT
    改为 error_reporting = E_ALL & ~E_NOTICE 即可解决前两条Strict Standards警告问题。

    至于后三条Warning，可能是UTF8文件的BOM引起，
    请确认session在开始前不能有任何字符输出，包括BOM。
    如果依然不能解决问题，请检查php.ini 文件中 session.save_path 目录是否可写。

  34. MiMi Says:
    2009.11.10 16:37

    把httpd.conf中的：
    #Include conf/extra/httpd-vhosts.conf
    去掉前面的注释，改为：
    Include conf/extra/httpd-vhosts.conf
    然后再httpd-vhosts.conf中添加虚拟主机。
    不要忘记把httpd-vhosts.conf中默认的虚拟主机配置注释掉。
