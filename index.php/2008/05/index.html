迭代和递归的区别

2008.05.25 // 1 Comment

迭代和递归各基于一种控制结构，都涉及到循环，都可无限进行 迭代利用循环，递归利用选择 迭代是循环求值，递归是调用本身 迭代使用循环结构，递归使用选择结构 迭代是当循环条件不满足时终止，递归是当满足基本条件时终止 迭代用计数器控制循环，不停地修改计数器的值，直到不满足条件为止 递归是逐渐逼近基本条件而终止，不断地对问题进行简化直到可以直接计算基本问题为止 经过这样的解释，是不是越来越迷糊？别急，慢慢体会
